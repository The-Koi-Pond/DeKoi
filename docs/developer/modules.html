Module guide

Where current DeKoi code lives, what it owns, and what depends on it.

Use this page before adding files or moving logic. It describes the current repository, not the old dashed project line and not future planned folders.

Top-Level Source Folders

Path Owner Do not put here Checks
src/app React bootstrap, providers, top-level state, storage sync, and view/action controller assembly. Product rules, storage adapters, engine imports, or raw runtime calls. pnpm check:frontend-boundaries
src/engine React-free records, contract leaf modules, pure actions, surface constants, and provider-neutral generation request assembly. React, browser helpers, feature UI, runtime adapters, Tauri, or raw provider transport. pnpm check:frontend-boundaries
src/features React surfaces and feature-owned workflows grouped under catalog, modes, navigation, runtime, and shell. Durable storage schemas, DB clients, host filesystem work, or duplicated engine rules. pnpm check:frontend-boundaries, pnpm build
src/runtime Storage contracts, collection adapters, app snapshots, bundle import/export, and legacy import normalization. React features, navigation, UI orchestration, generation UI flows, or raw Tauri packages. pnpm check:storage-contracts
src/shared Generic UI primitives, browser helpers, React helpers, and typed API wrappers under src/shared/api. Feature-specific workflows or concrete product state. pnpm check:frontend-boundaries
src-tauri Desktop host commands, app-data collection storage, native dialogs, provider secrets, and desktop runtime dispatch. React UI concerns or TypeScript product rules. pnpm check:desktop-contracts, pnpm check:rust

Feature Owners

Path Owns Primary dependencies
src/features/catalog Companion, Persona, Lorebook, and Connection editor surfaces plus record action hooks. src/engine/catalog/*-actions.ts, mode action modules under src/engine/modes, and focused shared API wrappers for provider checks and secrets.
src/features/modes/messenger Messenger thread UI, message send flow, edit/delete message controls, and Messenger thread action hooks. Messenger engine helpers, src/features/runtime/generation, shared chat composer UI.
src/features/modes/roleplay Roleplay thread UI, entry send flow, edit/delete entry controls, and Roleplay thread action hooks. Roleplay engine helpers, src/features/runtime/generation, shared chat composer UI.
src/features/modes/shared Mode-neutral chat composer, message time helpers, and generation delay helpers. Concrete modes may import it. It must not import concrete mode packages.
src/features/navigation Navigation context and typed nav state/action contracts. Feature owners receive narrow nav props instead of importing the full navigation context.
src/features/runtime/generation React-free generation workflows, provider generation, shared provider failure formatting, Messenger and Roleplay generation orchestration. Engine generation builders, shared API wrappers, provider connection records.
src/features/runtime/storage Initial app-storage loading, runtime target URL workflows, bundle previews, and import/export workflow wrappers. The lower src/runtime bridge and focused shared API wrappers.
src/features/runtime/ripples React-free Ripple state mutation workflows. Ripple engine records/actions. A routed Ripple editor surface does not exist yet.
src/features/shell Pond shell, Bank, Shoal, Waterline, Tide, Pond home sections, and Pond Care settings/data UI. Navigation props, feature runtime workflows, catalog/mode public entrypoints.

Runtime Storage Modules

Path Owns

src/runtime/storage/collections Per-collection normalizers and repository adapters.
src/runtime/storage/app-storage-snapshot.ts App-wide load/save/replace orchestration and assembly of split transcript records.
src/runtime/storage/app-storage-collection-projection.ts Dirty-collection signatures, counts, and source projection for split transcript storage.
src/runtime/storage/bundles DeKoi bundle import/export, desktop file bundle flow, desktop storage bundle flow, and one-way legacy import.
src/runtime/storage/storage-repository*.ts Repository contract and factory. Future database-backed storage belongs behind this shape.

Placeholder Or Not Implemented Yet
