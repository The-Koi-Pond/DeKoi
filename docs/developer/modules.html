Module guide

Where code lives, what it touches, and what depends on it.

Use this page before adding files or moving logic. The right home is the one that makes the dependency direction obvious.

Top-Level Source Folders

Path Owner Touches Impact area when changed
src/app React bootstrap and shell. App providers, startup effects, layout shell. Whole UI startup, provider wiring, modal/shell behavior.
src/features Feature-owned React UI. Feature hooks, components, UI stores, feature API facades. The feature's screens and user workflows.
src/shared Frontend shared layer. Generic UI, browser helpers, shared stores, runtime adapters. Every feature that imports the shared primitive.
src/engine React-free product engine. Domain rules, modes, generation, helper modules, repositories, contracts. Product behavior and every UI path using that engine service.
src-tauri Tauri host, hostable Rust runtime, and Rust capabilities. Commands, HTTP server/dispatch, state, storage, assets, provider transport, integrations. Native execution, remote runtime behavior, data persistence, provider calls, bundling.

Shared API Modules

Files in src/shared/api are the bridge from frontend code to embedded Tauri commands and the configured hostable Rust HTTP runtime. They should be typed and narrow. If business logic starts to grow here, move it into the TypeScript engine or the owning feature. New feature code should call these wrappers instead of importing tauri-client.ts, @tauri-apps/api, or calling the remote runtime with raw fetch directly.

Adapter Boundary
api-errors.ts Shared API error and JSON repair request shapes.
assets-api.ts Backgrounds, fonts, media assets, and asset browser commands.
avatar-api.ts Companion/persona/NPC avatar upload and resolution commands.
download-payload.ts and file-payload.ts Browser payload helpers for import/export and local file transfer.
export-api.ts Character, persona, lorebook, prompt, and profile export commands.
image-generation-api.ts Image generation, owner-typed Companion/persona sprite commands, gallery, and avatar generation commands.
import-api.ts legacy and SillyTavern import commands plus bulk import channels.
integration-gateway.ts, integration-utility-api.ts, tts-api.ts, and translation-api.ts TTS and translation command wrappers, Spotify gateway ports, knowledge sources, and connection utilities.
llm-api.ts LLM complete, stream, cancel, models, and connection tests.
local-file-api.ts Managed local file URL conversion and asset protocol helpers.
pond-guide-api.ts Pond Guide command wrapper.
profile-api.ts Profile export/import and profile file operations.
settings-assets-api.ts Settings-facing background and font commands.
storage-api.ts Typed entity CRUD, duplication, Messenger helpers, and generic storage access.
remote-runtime.ts Remote runtime URL parsing, command allowlist, JSON invoke fetch, SSE LLM streaming, and remote stream cancellation.
tauri-client.ts Single invoke/listen helper layer that chooses embedded Tauri by default and remote HTTP for allowlisted commands when Remote Runtime URL is configured.
ripples-snapshot-api.ts Ripple snapshot save/read commands for game and Classic state history.

Rust Command Modules

src-tauri/src/commands/storage.rs registers focused implementation modules and thin command facades. Keep new commands in the closest capability group.

Hostable runtime support belongs in http_dispatch.rs for JSON command reuse or http_server.rs for dedicated routes such as LLM SSE streaming. Do not create a parallel command implementation only for HTTP.

Group Owns
admin, keepers, custom_tools Admin resets, Keeper records, custom tool execution, helper memory, cadence, and patch/toggle commands.
bot_browser, ripple_snapshots Bot browser proxy commands and Ripple snapshot persistence.
companions, Messenger, prompts, knowledge Creative records, Messenger memory/notes/branches/swipes, prompt data, and knowledge source persistence.
avatars, backgrounds, fonts, media_assets, images, lorebook_images, media_uploads, sprites Managed media, asset paths, image generation, uploads, sprite cleanup, and asset browser operations.
exports, imports, profile legacy profile packages, SillyTavern imports, export payloads, ZIP handling, and profile asset migration.
generation, llm, http, translation Provider transport, streaming, model lists, safe fetches, and translation calls.
integrations Spotify, TTS, OAuth callback, and DJ guide playlist helpers.
pond-guide Pond Guide's helper-runtime-backed prompt command and read-only creative library tool.

Before moving code, check whether downstream imports should point to a public owner file, a lower-layer helper, or a capability port.
